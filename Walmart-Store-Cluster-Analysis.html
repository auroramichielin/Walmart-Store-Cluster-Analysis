// 01INTRODUZIONE ALLA CLUSTER ANALYSIS

Che cos'è la Cluster Analysis?

La Cluster Analysis è una tecnica statistica di classificazione senza supervisione. È un metodo tipicamente esplorativo: non esistono categorie predefinite, e l'obiettivo è riconoscere gruppi che appaiono con «naturalezza» nelle osservazioni. Le classificazioni assumono valenza interpretativa solo quando nei dati siano realmente presenti strutture di gruppo.

// OBIETTIVO STATISTICO

Considerare un gran numero di unità statistiche e creare gruppi distinti che contengono unità simili in base a tutte le variabili considerate, ovvero massimizzare l'omogeneità interna dei gruppi. Geometricamente: cercare le discontinuità nello spazio dei dati.

SUPERVISIONATO
Quando lo stato è noto
Si dispone già delle informazioni sullo stato dell'acquirente. Si usa questa informazione per la segmentazione, oppure la si ignora per verificarla a posteriori.
NON SUPERVISIONATO ← il nostro caso
Quando lo stato non è noto
Nella maggior parte dei casi non si dispone dello stato dell'acquirente o si vuole ampliare le quote di mercato: occorre procedere in modalità non supervisionata.

I 3 criteri di raggruppamento

HIERARCHICAL
Cluster Gerarchica
I dati vengono raggruppati gerarchicamente. L'output è un dendrogramma. Non richiede K a priori. → Ward
CENTROID-DISTANCE
Cluster non Gerarchica
Il numero di cluster è pre-definito. Ogni punto è assegnato al cluster il cui centroide è più vicino. → K-Means
DENSITY-BASED
Cluster basata su densità
Definisce i cluster come regioni densamente collegate. Identifica outlier automaticamente. → DBSCAN

// 02K-MEANS — CENTROID-DISTANCE-BASED

Come funziona il K-Means?

// PROCEDURA ITERATIVA · MINIMIZZAZIONE D_W · CONVERGENZA

Il K-Means è un algoritmo centroid-distance-based: il numero K di cluster è fissato a priori, e ogni punto viene assegnato al cluster il cui centroide è più vicino. L'obiettivo è minimizzare la devianza within D_W — la distanza euclidea tra ogni unità e il proprio centroide — garantendo la massima coesione interna.

I 3 passi dell'algoritmo — interattivo

Dati — nessuna etichetta
Abbiamo n punti nello spazio delle features senza informazioni sui gruppi di appartenenza.

Prima di applicare K-Means il codice esegue:
1. Aggregazione per store (media settimanale)
2. Standardizzazione Z-score con StandardScaler()Nel codice: K = 4, n_init=20, random_state=42.
Passo 1 — Centroidi iniziali (semi)
Si scelgono K poli iniziali. I criteri possibili sono:

· Le prime K unità
· Campione casuale (nel nostro codice: k-means++)
· K unità prestabilite già separate

Il codice usa n_init=20: 20 run con semi diversi, si tiene il migliore.
Passo 2 — Assegnazione al centroide più vicino
Ogni punto viene assegnato al cluster il cui centroide è più vicino:Se la distanza minima NON corrisponde al cluster attuale, il punto viene riassegnato e i centroidi coinvolti vengono ricalcolati.
Passo 3 — Ricalcolo dei centroidi
Per ogni cluster, il nuovo centroide è la media di tutti i punti assegnati:Le frecce mostrano lo spostamento dei centroidi.
Si torna al Passo 2 finché i centroidi non si spostano più.
Convergenza — Criterio di arresto
L'algoritmo si ferma quando:

· Nessun punto cambia cluster tra un'iterazione e l'altra
· Raggiunto il numero massimo di iterazioni
· Distanza tra centroidi < soglia

Il risultato minimizza la D_W — ma è un ottimo locale.

// 03CLUSTERING GERARCHICO — METODO DI WARD

Come funziona il Clustering Gerarchico?

// AGGREGATIVO · MATRICE DISTANZE · DENDROGRAMMA · WARD

Il Clustering Gerarchico Aggregativo parte con un oggetto per cluster e fonde iterativamente le classi più simili finché non ne rimane una sola. L'output è un dendrogramma. Non richiede K a priori.

// PROCEDURA GENERALE

1. Si calcola una matrice simmetrica di distanze tra tutte le unità.
2. Le due classi con distanza minima vengono fuse.
3. Si ricalcola la distanza tra la nuova classe e tutte le altre.
→ I diversi algoritmi differiscono nel punto 3.

I metodi di legame

Legame Singolo
d(AD,B) = min[d(A,B); d(D,B)]
Produce "catene" di oggetti. Non appropriato se l'errore di campionamento è elevato.
Legame Completo
d(AD,B) = max[d(A,B); d(D,B)]
Produce classi compatte e "dilata" lo spazio attorno ad esse.
Legame Intermedio
d(AD,B) = mean[d(A,B); d(D,B)]
Maggiore probabilità di riconoscere le partizioni "naturali".
Metodo di Ward ← usato nel codice
Aggrega la coppia con il minor incremento della varianza within (D_W). Non invariante per trasformazioni dei dati.
// METODO DI WARD — FORMULA

Aggrega ad ogni iterazione la coppia con minor incremento della varianza within, assicurando la maggiore coesione interna.

Come si legge il Dendrogramma?

Sull'asse Y troviamo la distanza Ward a cui avviene ogni fusione. Per scegliere K=4, si "taglia" il dendrogramma dove le altezze di fusione fanno un salto grande — intercettando esattamente 4 rami.

// 04DBSCAN — DENSITY-BASED SPATIAL CLUSTERING

Come funziona DBSCAN?

// ε-NEIGHBORHOOD · CORE POINTS · BORDER POINTS · OUTLIER

DBSCAN raggruppa i punti in alta densità e contrassegna come outlier i punti in regioni a bassa densità. Non richiede K a priori e trova cluster di forma arbitraria.

// I DUE PARAMETRI

ε (epsilon) = raggio del quartiere del punto p. Tutti i punti nel raggio ε da p: Nε(p) = {q ∈ D | dist(p,q) ≤ ε}
MinPts = numero minimo di punti in Nε(p) affinché p sia un core point.

I 3 tipi di punti

CORE POINT
Ha almeno MinPts punti nel suo ε-Neighborhood. È il "nucleo" del cluster. Ogni oggetto nel cluster è raggiungibile da un core point.
BORDER POINT
Non è core ma si trova nel quartiere di un core point. È sul bordo — fa parte del cluster ma non lo espande.
OUTLIER (NOISE)
Non raggiungibile da nessun core point. Etichettato come -1. Nel nostro caso: 4 store anomali.
// PSEUDOCODICE
FOR EACH x ∈ D IF x non appartiene a nessun cluster IF x è un core-object S_x = {y | y raggiungibile da x} ELSE N = N ∪ {x} // x è outlier → etichetta -1

Quando usare quale algoritmo?

K-MEANS
Quando si vuole un algoritmo veloce e si ha un'idea del numero di cluster. Tende a produrre cluster sferici di dimensione simile.
WARD
Quando si vuole esplorare la struttura senza conoscere K. Il dendrogramma permette una lettura visiva di tutte le partizioni possibili.
DBSCAN
Quando i dati hanno forma arbitraria e si vogliono identificare outlier automaticamente. Non funziona bene con densità variabile.

Il codice Python completo

Il codice implementa tutti e 3 gli algoritmi su dati reali (45 store Walmart). Di seguito il sorgente completo con annotazioni.

projectAI&ML.py Python 3.11 
import warnings
warnings.filterwarnings("ignore")

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

from sklearn.preprocessing import StandardScaler
from sklearn.cluster import KMeans, AgglomerativeClustering, DBSCAN
from sklearn.metrics import silhouette_score

from scipy.cluster.hierarchy import linkage, dendrogram

# ============================================================
# CONFIGURAZIONE
# ============================================================

DATASET = "Walmart_Store_sales.csv"
K = 4

# ============================================================
# CARICAMENTO DATI
# ============================================================

print("\n==============================")
print("CARICAMENTO DATASET")
print("==============================")

df = pd.read_csv(DATASET)

store_profile = (
    df.groupby("Store")
      .agg(
          sales=("Weekly_Sales", "mean"),
          temp=("Temperature", "mean"),
          fuel=("Fuel_Price", "mean"),
          cpi=("CPI", "mean"),
          unemp=("Unemployment", "mean")
      )
      .reset_index()
)

print(f"Numero store: {len(store_profile)}")

# ============================================================
# STANDARDIZZAZIONE
# ============================================================

FEATURES = [
    "sales",
    "temp",
    "fuel",
    "cpi",
    "unemp"
]

scaler = StandardScaler()
X = scaler.fit_transform(store_profile[FEATURES])

print("\nStandardizzazione completata.")

# ============================================================
# SILHOUETTE ANALYSIS
# ============================================================

print("\n==============================")
print("SILHOUETTE ANALYSIS")
print("==============================")

K_range = range(2, 11)
silhouettes = []

for k in K_range:

    km = KMeans(
        n_clusters=k,
        n_init=20,
        random_state=42
    )

    labels = km.fit_predict(X)

    sil = silhouette_score(X, labels)

    silhouettes.append(sil)

    print(f"K={k} -> Silhouette={sil:.4f}")

plt.figure(figsize=(8,5))

plt.plot(
    list(K_range),
    silhouettes,
    marker="o",
    linewidth=2
)

plt.axvline(
    K,
    linestyle="--",
    label=f"K scelto = {K}"
)

plt.title("Silhouette Analysis")
plt.xlabel("Numero Cluster")
plt.ylabel("Silhouette Score")
plt.grid(alpha=0.3)
plt.legend()

plt.tight_layout()
plt.savefig("1_silhouette_analysis.png", dpi=150)
plt.show()

# ============================================================
# K-MEANS
# ============================================================

print("\n==============================")
print("K-MEANS")
print("==============================")

kmeans = KMeans(
    n_clusters=K,
    n_init=20,
    random_state=42
)

km_labels = kmeans.fit_predict(X)

km_silhouette = silhouette_score(
    X,
    km_labels
)

print(f"Silhouette Score: {km_silhouette:.4f}")

cluster_sizes_km = (
    pd.Series(km_labels)
    .value_counts()
    .sort_index()
)

print("\nStore per cluster:")

for cluster, size in cluster_sizes_km.items():
    print(f"Cluster {cluster}: {size}")

# Scatter KMeans

plt.figure(figsize=(8,6))

for c in range(K):

    mask = km_labels == c

    plt.scatter(
        store_profile["sales"][mask],
        store_profile["cpi"][mask],
        s=80,
        label=f"Cluster {c}"
    )

plt.title("K-Means Clustering")
plt.xlabel("Average Sales")
plt.ylabel("Average CPI")
plt.legend()

plt.tight_layout()
plt.savefig("2_kmeans_scatter.png", dpi=150)
plt.show()

# ============================================================
# HIERARCHICAL CLUSTERING
# ============================================================

print("\n==============================")
print("HIERARCHICAL (WARD)")
print("==============================")

hc = AgglomerativeClustering(
    n_clusters=K,
    linkage="ward"
)

hc_labels = hc.fit_predict(X)

hc_silhouette = silhouette_score(
    X,
    hc_labels
)

print(f"Silhouette Score: {hc_silhouette:.4f}")

cluster_sizes_hc = (
    pd.Series(hc_labels)
    .value_counts()
    .sort_index()
)

print("\nStore per cluster:")

for cluster, size in cluster_sizes_hc.items():
    print(f"Cluster {cluster}: {size}")

# Dendrogramma

Z = linkage(
    X,
    method="ward"
)

plt.figure(figsize=(14,6))

dendrogram(
    Z,
    labels=[f"S{s}" for s in store_profile["Store"]],
    leaf_font_size=8
)

plt.title("Dendrogramma Ward")
plt.ylabel("Distanza")

plt.tight_layout()
plt.savefig("3_dendrogramma_ward.png", dpi=150)
plt.show()

# ============================================================
# DBSCAN
# ============================================================

print("\n==============================")
print("DBSCAN")
print("==============================")

db = DBSCAN(
    eps=1.8,
    min_samples=3
)

db_labels = db.fit_predict(X)

n_clusters_db = (
    len(set(db_labels))
    - (1 if -1 in db_labels else 0)
)

n_outliers = list(db_labels).count(-1)

print(f"Cluster trovati: {n_clusters_db}")
print(f"Outlier trovati: {n_outliers}")

cluster_sizes_db = (
    pd.Series(db_labels)
    .value_counts()
    .sort_index()
)

print("\nStore per cluster:")

for cluster, size in cluster_sizes_db.items():

    if cluster == -1:
        print(f"Outlier: {size}")
    else:
        print(f"Cluster {cluster}: {size}")

# Silhouette DBSCAN

if n_clusters_db > 1:

    mask = db_labels != -1

    db_silhouette = silhouette_score(
        X[mask],
        db_labels[mask]
    )

else:

    db_silhouette = 0

print(f"\nSilhouette Score: {db_silhouette:.4f}")

# Scatter DBSCAN

plt.figure(figsize=(8,6))

for lbl in sorted(set(db_labels)):

    mask = db_labels == lbl

    if lbl == -1:

        plt.scatter(
            store_profile["sales"][mask],
            store_profile["cpi"][mask],
            marker="X",
            s=120,
            label="Outlier"
        )

    else:

        plt.scatter(
            store_profile["sales"][mask],
            store_profile["cpi"][mask],
            s=80,
            label=f"Cluster {lbl}"
        )

plt.title("DBSCAN")
plt.xlabel("Average Sales")
plt.ylabel("Average CPI")
plt.legend()

plt.tight_layout()
plt.savefig("4_dbscan_scatter.png", dpi=150)
plt.show()

# ============================================================
# CONFRONTO FINALE
# ============================================================

comparison = pd.DataFrame({
    "Algoritmo": [
        "K-Means",
        "Ward",
        "DBSCAN"
    ],
    "Silhouette": [
        km_silhouette,
        hc_silhouette,
        db_silhouette
    ]
})

print("\n==============================")
print("CONFRONTO FINALE")
print("==============================")
print(comparison.round(4))

plt.figure(figsize=(7,4))

plt.bar(
    comparison["Algoritmo"],
    comparison["Silhouette"]
)

plt.title("Confronto Silhouette")
plt.ylabel("Silhouette Score")

plt.tight_layout()
plt.savefig("5_confronto_algoritmi.png", dpi=150)
plt.show()

# ============================================================
# FILE GENERATI
# ============================================================

print("\n==============================")
print("FILE GENERATI")
print("==============================")

print("1_silhouette_analysis.png")
print("2_kmeans_scatter.png")
print("3_dendrogramma_ward.png")
print("4_dbscan_scatter.png")
print("5_confronto_algoritmi.png")
